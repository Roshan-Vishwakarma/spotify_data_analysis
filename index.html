Project Details

Project Requirements

This project involved extensive data analysis and visualization using R programming language and related libraries. The main objectives included exploring patterns, trends, and insights within the Spotify music dataset obtained from the official Spotify Developer Website.

Data Source

The dataset utilized in this project was obtained from the Spotify Developer Website.

Libraries Used

The following R libraries were utilized for data analysis, visualization, and processing:

  • dplyr: Used for data manipulation and summarization tasks.
  • ggplot2: Utilized for creating a wide range of visualizations (bar charts, line charts, etc.) to illustrate various insights derived from the dataset.
  • tidyverse: A collection of R packages including dplyr and ggplot2, among others, facilitating data manipulation and visualization in a consistent manner.
  • tidyr: Used for data tidying tasks, helping reshape data frames to facilitate analysis and visualization.
